Consigna:

Escribe un bucle do while que sume los números ingresados por el usuario y los imprima en la consola, deteniéndose cuando el usuario ingrese el número cero.

Código utilizado:

let numero;
let suma = 0;
do {
numero=parseFloat(prompt ("Ingrese un número para sumar"));
if (!isNaN(numero)){
suma = suma+numero;
}
}
while (numero != 0)
alert ("La suma es " +suma);
